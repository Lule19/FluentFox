# FluentFox - Foreign Language School

I developed a modern and responsive website for FluentFox, a foreign language school based in Aranđelovac, Serbia. The site represents a complete digital solution that enables prospective students to easily find all necessary information about the school and courses.

Key features:

Overview of all available foreign language courses
Detailed information about school location and working hours
Contact form for inquiries and applications
Responsive design adapted for all devices (desktop, tablet, mobile)
Modern UI/UX implementation for optimal user experience
Interactive navigation and content presentation
Technologies: React.js, responsive design, modern web standards

# FluentFox - Škola stranih jezika

Razvio sam moderan i responzivan veb sajt za FluentFox, školu stranih jezika sa sedištem u Aranđelovcu. Sajt predstavlja kompletno digitalno rešenje koje omogućava potencijalnim polaznicima da lako pronađu sve potrebne informacije o školi i kursevima.

Ključne funkcionalnosti:

Pregled svih dostupnih kurseva stranih jezika
Detaljan prikaz informacija o lokaciji škole i radnom vremenu
Kontakt forma za upite i prijave
Responzivan dizajn prilagođen svim uređajima (desktop, tablet, mobilni)
Moderna UI/UX implementacija za optimalno korisničko iskustvo
Interaktivna navigacija i prezentacija sadržaja
Tehnologije: React.js, responzivni dizajn, moderni web standardi
